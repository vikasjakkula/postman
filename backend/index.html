Postman + Fetch playground

Backend: http://localhost:5000. Every button below runs a fetch() to the Express server and prints the JSON response. The exact fetch code for each one is shown so you can copy it into your React projects. Try the same URLs in Postman too.

1. GET — read all tasks

Optional query params: ?done=true / ?done=false / ?search=postman

2. GET — read one task (route param)

3. POST — create a task (JSON body)

4. PUT — replace a task (send ALL fields)

5. PATCH — update part (toggle done)

6. DELETE — remove a task
